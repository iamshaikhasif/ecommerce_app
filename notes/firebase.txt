Setup Firebase using CLI

1. Install Node Js
2. npm install -g firebase-tools 
3. dart pub global activate flutterfire_cli
4. After Succssing this command you'll find the warining to set the path in your enviroment. Set the path in you enviroment
5. firebase login
6. flutterfire configure
7. select your project and done your firebase is setup now
